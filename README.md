# Springlet

A simple CLI tool to generate Spring Boot projects directly from the terminal.

Built to avoid opening the browser for Spring Initializr — just run a command, pick dependencies, and you're ready.

---

## Install

curl -sL https://raw.githubusercontent.com/AshwanthReddy-exe/Springlet/main/install.sh | bash

---

## Usage

springlet init

---

## What it does

- Interactive dependency search and selection  
- Generates Spring Boot projects using Spring Initializr  
- Works entirely from the terminal  

---

## Requirements

- Java 17+  
- Internet connection  

---

## Why Springlet?

Sometimes you just want to quickly bootstrap a Spring project without opening a browser.

Springlet makes that fast and simple.

---

## Notes

- Projects are generated using Spring Initializr  
- This is a minimal CLI tool focused on simplicity
